# IV_jump-detector-for-options-pricing

Leveraging Discrete Distributional Derivatives to Assess Strike Prices via Jump Detection Techniques

This project introduces a Jump Detection technique by leveraging the discrete analog of distributional derivatives. The goal was to assess changes in implied volatility relative to the strike prices by affirming fair prices and detecting possible mispriced/misordered call prices via sudden jumps. Using second-order finite-differences of IV between neighboring strike prices, a six-step procedure is employed to demonstrates the utility of the Jump Detection Technique with real stock options data.
