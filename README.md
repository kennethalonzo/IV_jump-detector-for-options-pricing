# IV_jump-detector-for-options-pricing

Locating Implied Volatility Jumps to Identify Anomalies in Strike Prices

Kenneth A. Anderson,  Ph.D.

This project introduces a Jump Detection technique by leveraging the discrete analog of distributional derivatives. The goal was to assess changes in implied volatility (IV) relative to the strike prices. The Jump Detector is able to affirm fair prices and detect possible mispriced/misordered call prices via sudden jumps. Using second-order finite-differences of IV between neighboring strike prices, a six-step procedure is employed to demonstrates the utility of the Jump Detection Technique with real stock options data.
