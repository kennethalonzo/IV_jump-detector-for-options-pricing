# IV_jump-detector-for-options-pricing

Locating Implied Volatility Jumps to Identify Anomalies in Strike Prices

Kenneth A. Anderson,  Ph.D.

This project introduces an implied volatility (IV) Jump Detection technique by leveraging the discrete analog of distributional derivatives. The goal was to assess changes in implied volatility (IV) relative to the strike prices. The Jump Detector is able to identify unexpected jumps in IV which may prompt further investigation for mispriced/misordered strike prices. Using second-order finite-differences of IV between neighboring strike prices, a six-step procedure is employed to demonstrate the utility of the Jump Detection Technique with real stock data.
